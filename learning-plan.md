# 个人学习计划

> 最后更新：2026-05-18

基于 [AI × Web3 School Handbook](https://aiweb3.school/zh/handbook/) 结构，
结合我（小铜）的基础和目标（开发 + 黑客松）制定的学习路径。

---

## 📅 整体路线

### 第一阶段：补齐基础（第 1–2 周）
快速扫清概念，不打无准备之仗。

**AI 基础路线：**
1. [LLM](https://aiweb3.school/zh/handbook/ai/llm/) — 大模型能做什么、不能做什么
2. [Prompt](https://aiweb3.school/zh/handbook/ai/prompt/) — 怎么跟模型对话
3. [Context](https://aiweb3.school/zh/handbook/ai/context/) — 模型"看得到"什么
4. [RAG](https://aiweb3.school/zh/handbook/ai/rag/) — 给模型喂外部知识
5. [Agent](https://aiweb3.school/zh/handbook/ai/agent/) — 让模型动手执行

**Web3 基础路线：**
1. [Network](https://aiweb3.school/zh/handbook/web3/network/) — 区块链基础概念
2. [Cryptography](https://aiweb3.school/zh/handbook/web3/cryptography/) — 哈希、密钥、签名
3. [Wallet](https://aiweb3.school/zh/handbook/web3/wallet/) — 钱包就是身份入口
4. [Smart Contract](https://aiweb3.school/zh/handbook/web3/smart-contract/) — 链上规则
5. [Account Abstraction](https://aiweb3.school/zh/handbook/web3/account-abstraction/) — Agent 权限的基础

### 第二阶段：进入交叉（第 3–4 周）
开始把两边的知识连起来。

重点章节（AI × Web3 Bridge）：
1. [Chain-aware Context](https://aiweb3.school/zh/handbook/bridge/chain-aware-context/)
2. [Web3 Tool Use](https://aiweb3.school/zh/handbook/bridge/web3-tool-use/)
3. [Agent Workflow](https://aiweb3.school/zh/handbook/bridge/agent-workflow/)
4. [Agent Wallet](https://aiweb3.school/zh/handbook/bridge/agent-wallet/)

### 第三阶段：动手做项目（持续进行）
参加黑客松，落地一个最小原型。

参考方向（[前沿探索](https://aiweb3.school/zh/handbook/tracks/agentic-commerce/)）：
- Agentic Commerce
- Wallet / Permission
- AI Security
- Governance

## 📊 每日节奏（3 小时）

| 时段 | 内容 | 时间 |
|------|------|------|
| 阅读 Handbook + 做笔记 | 概念理解 | ~1.5h |
| 动手实验 / 跟教程跑代码 | 实践巩固 | ~1h |
| 写 daily note + 打卡 | 沉淀复盘 | ~0.5h |
